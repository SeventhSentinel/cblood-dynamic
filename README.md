# CBlood-Dynamic
A mod for UZDoom that corrects monster blood colors

A fork of the excellent [CBlood](https://github.com/nashmuhandes/cblood) by Nash Muhandes

License: MIT

## Supported Games
- Doom, Doom II, Final Doom
- Chex Quest
- Hexen

## Supported Mapsets
- Ancient Aliens
- Back to Saturn X
- Baculus
- DBP61
- Disjunction
- Eviternity & Eviternity II
- Exomoon
- Fractured Worlds
- Infection
- Necromantic Thirst
- Rowdy Rudy's Revenge & Rowdy Rudy II
- Scientist 2023
- Struggle
- Sunder
- Tangerine Nightmare

### Compatibility
- CBlood-Dynamic is ZScript-based. I have only tested it in UZDoom.
- Blood graphics will look best in paletted rendering modes.

### Installation
1. [Download the latest release](https://github.com/SeventhSentinel/cblood-dynamic/releases/latest).
2. Add to your [autoload](https://zdoom.org/wiki/How_to_autoload_files).
